# 新手教程 + 副本经济重设计 实现方案

## 一、新手教程系统

### 设计目标
- 首次进入游戏时自动触发
- 6步引导，每步高亮对应UI元素
- 视觉鲜明：半透明遮罩 + 高亮框 + 箭头指引 + 动画文字
- 可跳过

### 教程步骤
1. **欢迎** - 展示游戏名称和角色，点任意继续
2. **状态栏** - 高亮顶部 Lv/Gold/Wave/Kills/Enemies
3. **英雄信息** - 高亮右侧角色面板（HP/MP/EXP条）
4. **技能释放** - 高亮底部技能按钮，提示点击释放
5. **移动塔位** - 高亮地图上的塔，提示点击移动
6. **副本系统** - 高亮副本按钮，提示金币用途

### 技术方案
- HTML/CSS覆盖层，z-index最高
- localStorage 存储 `warcraft_tutorial_done` 标记
- 使用绝对定位 + box-shadow 实现高亮效果
- CSS动画实现箭头跳动和文字渐入

## 二、副本经济重设计

### 当前问题
- 副本金币消耗太低（Lv.1仅需5金）
- 副本难度与主线脱节
- 金币没有明确消费出口

### 经济闭环设计
```
打怪获取金币 → 金币挑战副本 → 副本产出大量经验 → 升级变强 → 挑战更高波次
     ↑                                                        ↓
     └──────────── 波次越高，打怪金币越多 ←────────────────────┘
```

### 难度设计原则
- Lv.1-3: 休闲级，新手能过，建立信心
- Lv.4-6: 挑战级，需要一定等级和装备
- Lv.7-9: 困难级，需要策略和操作
- Lv.10: 地狱级，需要满配英雄
