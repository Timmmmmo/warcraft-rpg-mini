# 魔兽RPG塔防 V2.0 - 需求设计文档

## 项目背景
参考魔兽争霸3 RPG地图"刷不完的怪"核心玩法，打造手机竖版塔防RPG游戏。

---

## 一、核心设计理念 (项目经理+产品经理讨论)

### 1.1 原版玩法解析
**"刷不完的怪"核心机制：**
- 单英雄站桩防守
- 英雄固定在塔位上自动攻击
- 怪物沿固定路线行进（不攻击英雄）
- 英雄通过击杀怪物获得经验和金币
- 金币用于升级英雄属性和技能
- 经验用于升级和转职
- 无限波次，挑战最高波次记录

**核心乐趣：**
- 转职系统的随机性（自动vs手动）
- 数值成长的爽快感
- 技能搭配的策略性
- 挑战极限波次的成就感

### 1.2 差异化设计
| 原版WC3 | 本游戏 |
|---------|--------|
| PC操作 | 手机竖屏适配 |
| 鼠标点击 | 触屏操作 |
| 固定视角 | 优化视觉效果 |
| 基础UI | 腾讯级UI品质 |

## 二、游戏系统设计

### 2.1 地图设计

```
┌─────────────────────────────┐
│                             │
│   [T1]                 [T2] │  ← 四角塔位
│                             │
│      ╭───────────────╮      │
│      │   内环路线    │      │  ← 内环怪物路线
│      │              │      │
│      │    [T5]      │      │  ← 中心塔位
│      │              │      │
│      ╰───────────────╯      │
│                             │
│   [T3]                 [T4] │  ← 四角塔位
│                             │
│      ╭═══════════════╮      │
│      │   外环路线    │      │  ← 外环怪物路线
│      ╰═══════════════╯      │
│                             │
└─────────────────────────────┘
```

**5个塔位：**
- T1: 左上角 (50, 100)
- T2: 右上角 (350, 100)
- T3: 左下角 (50, 500)
- T4: 右下角 (350, 500)
- T5: 中心点 (200, 300)

**怪物路线：**
- 内环：沿地图内圈矩形路径
- 外环：沿地图外圈矩形路径

### 2.2 英雄系统

**单英雄机制：**
- 游戏开始时只有一个英雄
- 英雄可以移动到任意塔位
- 不同塔位适合不同阶段/职业

**基础属性：**
- HP: 生命值
- MP: 魔法值
- ATK: 攻击力
- DEF: 防御力
- ATK_SPD: 攻击速度
- ATK_RANGE: 攻击范围（重要！）
- CRIT: 暴击率

**攻击范围与塔位选择策略：**
| 职业类型 | 攻击范围 | 推荐塔位 |
|----------|----------|----------|
| 近战战士 | 短(80) | T5中心（覆盖内环） |
| 远程弓手 | 长(150) | T1-T4四角（覆盖外环） |
| 法师 | 中(120) | 任意位置 |

### 2.3 转职系统（核心玩法）

**转职机制 - 随机vs选择：**

```
         ┌─────────────────┐
         │   Lv.10 达成    │
         └────────┬────────┘
                  │
         ┌────────┴────────┐
         │                 │
    ┌────▼────┐      ┌─────▼─────┐
    │ 自动转职 │      │ 手动转职  │
    │ +20%属性 │      │ +10%属性  │
    │ 随机职业 │      │ 自选职业  │
    └─────────┘      └───────────┘
```

**转职树：**

```
战士(Warrior) ─┬─→ 剑圣(BladeMaster) ─→ 剑神(SwordGod)
              ├─→ 山丘(MountainKing) ─→ 泰坦(Titan)
              └─→ 血法(BloodMage) ─→ 炎魔(Inferno)

弓手(Archer) ─┬─→ 风行(WindRunner) ─→ 疾风(Gale)
             ├─→ 暗猎(ShadowHunter) ─→ 暗影之王(ShadowLord)
             └─→ 守护(Keeper) ─→ 自然之灵(NatureSpirit)

法师(Mage) ─┬─→ 冰霜(FrostMage) ─→ 冰皇(FrostEmperor)
           ├─→ 火焰(FireMage) ─→ 凤凰(Phoenix)
           └─→ 雷电(StormMage) ─→ 雷神(ThunderGod)
```

**转职变化：**
- 一转(10级): 技能变化+属性增强
- 二转(25级): 大招升级+终极形态
- 三转(40级): 终极技能+外观变化

### 2.4 技能系统

**技能槽位：**
- Q: 小技能1（低消耗，短CD）
- W: 小技能2（中消耗，中CD）
- E: 被动技能/辅助
- R: 大必杀（高消耗，长CD）

**转职后技能变化示例（战士系）：**

| 阶段 | Q技能 | W技能 | E技能 | R技能 |
|------|-------|-------|-------|-------|
| 战士 | 普攻强化 | 旋风斩 | 生命回复 | 全屏伤害 |
| 剑圣 | 剑气斩 | 三连斩 | 暴击提升 | 剑刃风暴 |
| 剑神 | 万剑归宗 | 剑雨 | 必杀一击 | 天外飞仙 |

### 2.5 怪物系统

**怪物类型：**
- 普通怪: 数量多，血量低
- 精英怪: 数量少，血量高，有特殊技能
- Boss: 每5波出现，超高血量

**怪物行进：**
- 怪物不攻击英雄
- 沿固定路线行进
- 到达终点消失（或扣生命值？待定）
- 击杀获得经验和金币

**怪物路线：**
- 内环: 周期短，怪物速度快
- 外环: 周期长，怪物速度慢但更肉

### 2.6 经济系统

**金币获取：**
- 击杀普通怪: 5-10金
- 击杀精英怪: 20-30金
- 击杀Boss: 100金

**金币用途：**
- 升级攻击力: 100→300→800→1500金
- 升级防御力: 80→250→600→1200金
- 升级攻速: 150→400→900金
- 升级范围: 200→500→1200金
- 学习技能: 各技能有不同价格

## 六、讨论要点 (军师+丞相)

### Q1: 英雄是否可以移动？
A: 可以。玩家可以随时移动英雄到不同塔位，这是策略点之一。

### Q2: 怪物到达终点的惩罚？
A: 扣除生命值(1点/Boss扣5点)，生命值归零游戏结束。

### Q3: 是否有多英雄？
A: V2版本只有单英雄，专注于转职系统的深度体验。

### Q4: 自动转职的随机池？
A: 根据当前基础职业的转职树随机，不会转到其他系。

### Q5: 内环外环的怪物差异？
A: 内环：速度快+血量低；外环：速度慢+血量高+奖励多。
