# 游戏设计文档

## 项目概述

魔兽RPG小游戏是一款基于Cocos Creator开发的轻量级RPG游戏。

## 核心系统

### 1. 角色系统
- 角色创建与选择
- 属性管理（HP、MP、ATK、DEF等）
- 等级与经验系统

### 2. 战斗系统
- 回合制战斗
- 技能释放
- 伤害计算

### 3. 任务系统
- 主线任务
- 支线任务
- 任务奖励

### 4. 装备系统
- 装备获取
- 装备强化
- 装备合成

### 5. 技能系统
- 技能学习
- 技能升级
- 技能释放

## 游戏流程

1. 游戏启动 → 主菜单
2. 创建/选择角色
3. 进入游戏世界
4. 完成任务、战斗、升级
5. 存档与退出

## 美术风格

- 2D 像素风格或卡通风格
- 魔兽主题美术资源

## 音效设计

- 背景音乐
- 战斗音效
- UI音效
